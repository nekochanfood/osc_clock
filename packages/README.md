# packages

## app
Desktop GUI application (SvelteKit + Tauri).

## build-utils
Rust build-time utilities used by other crates.

## core
Core Rust library + CLI (OSC sender for VRChat).

## docs
Documentation website (Docusaurus).

## horloger
Unity package for creating clocks that work with OSC Clock.

## shared
Shared Rust crate used across packages.
